on définit un 'canvas' de 150px de largeur et de hauteur on va utiliser la méthode arc()
arc(x, y, rayon, angleInitial, angleFinal, antihoraire)
x et y indiquent où se trouve le centre du cercle. on indique le rayon antihoraire : si c'est à true le cercle est déssiné dans le sens inverse des aiguilles d'une montre. 

       ctx.beginPath(); //on commence à dessiner
       ctx.arc(75,75,50,0,Math.PI*2,true); //
       ctx.stroke();// on dessine un trait
